Acacia

Accessibility

Acacia aims to provide a clear, native macOS PDF workflow with labelled controls, keyboard-reachable actions, visible focus states, and system accessibility support where possible.

Supported Practices

  • Native macOS windows, menus, text inputs, and PDF rendering surfaces.
  • Accessible names for primary actions such as search, navigation, export, annotations, and settings.
  • Keyboard support for common library and viewer workflows.
  • High-contrast interface states and visible focus outlines on the website.

Feedback

If you encounter an accessibility issue, email support@benebsworth.com with the workflow, macOS version, and assistive technology you are using.
