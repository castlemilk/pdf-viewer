Acacia

Support

Acacia is a local-first macOS PDF workspace. It does not require an account for the core app and does not upload your PDF documents to a cloud service.

Contact

Email support@benebsworth.com for product support, privacy requests, or accessibility feedback.

Common Tasks

  • Open a PDF from the library or system file picker.
  • Add tags, collections, favourites, and reading progress from the library.
  • Use thumbnails, search, zoom, metadata, and page controls in the viewer.
  • Add highlights, notes, comments, drawings, signatures, and bookmarks.
  • Export annotated copies, page images, extracted text, or Markdown where available.
  • Use compare mode to review two document versions side by side.

Troubleshooting

  • If a PDF opens blank, confirm the original file still exists and is accessible from macOS.
  • If export output is missing, check the destination selected in the save dialog.
  • If annotation tools do not respond, reopen the document and confirm the PDF is not password protected.
  • If testing automation, grant Accessibility and Automation permissions in System Settings.
